# 视频解析工具

一款基于 Electron 开发的多平台视频无水印解析下载工具,支持抖音、快手、小红书、B站等主流平台。

## 功能特性

- 🎬 **多平台支持** - 支持抖音、快手、小红书、B站等主流视频平台
- 🚫 **无水印下载** - 自动解析并下载无水印高清视频
- 📦 **批量下载** - 支持批量解析和下载多个视频
- 🎨 **现代化界面** - 简洁美观的用户界面,操作流畅
- ⚡ **高性能** - 优化的下载引擎,支持多线程并发下载
- 🖼️ **图片处理** - 自动转换图片格式,优化存储空间
- 📝 **下载管理** - 实时显示下载进度和任务状态
- 🔧 **灵活配置** - 自定义下载路径、文件命名等设置
- 🎯 **单实例运行** - 防止重复启动,优化系统资源

## 系统要求

- Windows 10 及以上版本
- 无需额外安装 Node.js 或浏览器环境

## 技术栈

- **框架**: Electron 36.0.0
- **语言**: JavaScript (ES6+)
- **图片处理**: Sharp 0.33.5
- **打包工具**: electron-builder 25.1.8
- **代码混淆**: javascript-obfuscator 4.1.1

## 开发指南

### 安装依赖

```bash
npm install
```

### 开发运行

```bash
npm start
```

### 构建打包

```bash
npm run build
```

打包后的文件位于 `dist` 目录。

## 使用说明

1. **复制链接** - 从目标平台复制视频分享链接
2. **粘贴解析** - 将链接粘贴到解析框中
3. **选择下载** - 选择需要下载的视频或图片
4. **开始下载** - 点击下载按钮,等待完成

## 配置说明

配置文件位于用户目录下的 `AppData/Roaming/video-parser-app/config.json`,支持以下配置项:

- `downloadPath` - 下载保存路径
- `fileNaming` - 文件命名规则
- `concurrentDownloads` - 并发下载数量
- `autoConvertImage` - 自动转换图片格式

## 性能优化

- 采用 IIFE 命名隔离,避免全局变量污染
- 优化内存占用,及时释放资源
- 多线程并发下载,提升下载速度
- 智能缓存机制,减少重复请求

## 安全性

- 代码混淆保护,防止逆向工程
- 无需开发者模式,保护用户隐私
- 本地化配置存储,数据安全可控
